üöÄ Technicalities

This project is a classic web app built around a UI ‚Üí API ‚Üí Database architecture.

The API layer is developed with Fastify and designed for flexibility, using Awilix for dependency injection (DI) management. DI is essential here, as it makes testing and local development easier (especially with Docker) and allows for greater architectural flexibility when swapping components.

On the backend, I rely heavily on AWS services, including CloudFront, Cognito, API Gateway, Lambda, ECS, S3, SQS, SNS, and DynamoDB, along with infrastructure essentials like VPCs, secrets management, and CloudFormation. All infrastructure is defined using CloudFormation YAML files. For local development I use Localstack to fake AWS services integration.

note: below there's a diagram of the AWS architecture for this project and how everything ties together.

On the frontend, the UI is built with Angular 19 and uses Ngrx Signal Store for state management, with RxJs handling asynchronous operations like image uploads and API calls. Although I have used frameworks like Angular Material in the past, this time I went with Tailwind CSS (even though I'm still using Angular Material CDK for stuff like dialogs). Other key libraries include Angular OIDC for Cognito OAuth authentication and Google Maps.
